How to convert SVG to PNG:

  1. Option 1 (Easiest): Right-click the image below → "Save Image As" → Save as PNG
  2. Option 2: Use an online converter like cloudconvert.com
  3. Option 3: Install ImageMagick and run:
    brew install imagemagick
    magick public/og-image.svg public/og-image.png

Save the PNG as: public/og-image.png
